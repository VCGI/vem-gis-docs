## What are ArcGIS Dashboards?

[ArcGIS Dashboards](https://doc.arcgis.com/en/dashboards/latest/get-started/what-is-a-dashboard.htm) are used to display static and real-time data using a variety of visualizations (e.g., maps, graphs, tables, indicators, etc.). They can focus on a specific use case or combine multiple data sources to support planning and decision-making before, during, and after emergency-related events.

## Key Features

- **Real-time Data Integration**: Live feeds from weather services, traffic monitoring, and emergency services
- **Interactive Mapping**: Multiple map configurations for different operational needs
- **Custom Widgets**: Specialized tools for incident tracking, resource management, and reporting
- **Arcade Expressions**: Dynamic symbology and data-driven visualizations
