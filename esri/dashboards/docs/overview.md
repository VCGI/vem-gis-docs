# Dashboards Overview

The following dashboards were developed for VEM to support planning and decision-making related to executing the [State Emergency Management Plan](https://vem.vermont.gov/plans/state).

## Current Dashboards
Multiple map views provide different perspectives on Vermont's emergency management landscape:

- **[Vermont FEMA Declarations Explorer](https://experience.arcgis.com/experience/cd391c42afc346a0b11a4e9ae7f70e77)**: This dashboard allows public users (e.g., municipalities, RPCs) to explore historical FEMA declarations in their respective counties. The data shared in this dashboard is pulled from two FEMA API endpoints and joined to a Vermont counties feature service.
