# Overview

The VEM COP Experience Builder application is organized into multiple pages, each designed for specific operational functions. This section provides detailed documentation for each page.

## Available Pages

### [Overview Page](overview-page.md)
Main dashboard providing a comprehensive view of current statewide conditions.

**Key Features:**

- Multi-map display with synchronized views
- Summary statistics and indicators
- Quick access to all operational functions
- Real-time data feeds

**Best Used For:** Initial situational awareness and daily briefings

---

### [Weather Page](weather-page.md)
Dedicated weather monitoring and forecasting interface.

**Key Features:**

- NWS alerts and warnings
- Radar imagery and loops
- Forecast data and models
- Weather station observations
- Historical weather data

**Best Used For:** Weather event preparation and monitoring

---

### [Fire Risk Page](fire-risk-page.md)
Fire danger assessment and wildfire monitoring.

**Key Features:**

- Fire danger ratings by region
- Active fire incidents
- Fire weather forecasts
- Resource availability
- Historical fire data

**Best Used For:** Fire season planning and wildfire response

---

### [VT Outages Page](vt-outages-page.md)
Power outage tracking and utility coordination.

**Key Features:**

- Real-time outage data by utility
- Customer impact statistics
- Town-level aggregation
- Outage duration tracking
- Restoration estimates

**Best Used For:** Power outage monitoring and public information

---

### [Mass Care Page](mass-care-page.md)
Shelter and mass care resource management.

**Key Features:**

- Emergency shelter locations
- Capacity and occupancy tracking
- Resource coordination tools
- Contact information

**Best Used For:** Shelter activation and mass care operations

## Page Navigation

### Accessing Pages
Pages are accessed via the navigation bar in the Vermont Emergency Management COP Experience Builder application. Click any page tab to switch views.

### Page Structure
Each page typically includes:

- **Header**: Page title and quick actions
- **Map View**: Interactive map with relevant layers
- **Side Panel**: Widgets for data display and interaction

## Mobile Access

!!! warning "Mobile Limitations"
    Some advanced features may have limited functionality on mobile devices. Desktop browsers are recommended for full operational use.
